# 🧠 뇌를 깨우는 30초 - YouTube Shorts 자동화

매일 자동으로 심리학/자기계발 YouTube Shorts를 생성하는 시스템입니다.

## ✨ 기능

- 🤖 **AI 스크립트 생성** - Gemini API로 매일 새로운 스크립트
- 🔊 **자동 음성 생성** - gTTS로 한국어/영어 나레이션
- 🎥 **배경 영상 자동 다운로드** - Pexels API (저작권 무료)
- 📝 **자동 자막 생성** - ASS 포맷 애니메이션 자막
- 🎬 **자동 영상 합성** - FFmpeg로 최종 영상 생성
- 📤 **YouTube 자동 업로드** - (옵션) YouTube Data API
- 📱 **텔레그램 알림** - 생성 결과 실시간 알림
- ⏰ **매일 자동 실행** - GitHub Actions 스케줄
